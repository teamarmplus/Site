# Client Consent Rules

- Client details are never shared with a trader automatically.
- The client chooses whether to be introduced to a suggested trader — consent is required first.
- SiteVerdict suggests suitable verified traders; it does not guarantee the trader's outcome.
- The trader is an independent business; SiteVerdict is not the builder/contractor.
- Client feedback is collected to help quality control and improve future matching.
- If the client is unhappy, SiteVerdict can review the issue and suggest another suitable option if one is available.
- Client details remain hidden until: T approves the match, a trader is selected, the client consents,
  the trader has accepted the agreement rules, and the privacy warning is acknowledged.

This is client-first matching. It is not a guarantee of any result, price, or quality.
